# 🐾 Animal Rescue AI System
An AI-powered emergency response platform for stray animal rescue.

## 🚀 Key Features
* **AI Verification:** Automatically identifies and verifies animals using TensorFlow.js (MobileNet).
* **Smart Filtering:** Rejects non-animal reports to prevent spam.
* **Live Analytics:** Real-time dashboard for tracking total, pending, and successful rescues.
* **GPS Integration:** Captures exact location coordinates for accurate rescue missions.

## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Frontend:** HTML5, CSS3, JavaScript
* **Database:** SQLite
* **Machine Learning:** MobileNet (Pre-trained Neural Network)

## 📋 How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the server: `python app.py`
3. Open `http://127.0.0.1:5001` in your browser.
